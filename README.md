# Collection of algorithms in Rust

**algorithms-rs** is a **learning / educational collection** of data structures and algorithms implemented in **Rust**.

This is a **library crate** (not a binary), created mainly for:
- personal study and better understanding of algorithms
- preparing for coding interviews
- practicing clean, idiomatic Rust code
- having a single place with tested and documented implementations

## Implemented

### Search
- Binary Search

### Sort 
- Selection sort

### Recursion 
- Sum of array
